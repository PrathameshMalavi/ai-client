Agent-to-Agent (A2A)

This merchant provides its own AI agent via an A2A (Agent-to-Agent) connection.

Instead of making direct API calls, this client forwards your messages to the merchant's agent. The merchant's agent handles the orchestration and replies with structured UI components for this client to render.
